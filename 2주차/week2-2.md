#Java의 반복문
1. For-loop
 
 - 형식 : For(초기값 ; 조건문; 증감)

 - 사용 예 : 1부터 10까지의 합을 더한다.

				*****************************
				*	For(i=1; i<=10; i++){   *
				*		sum+=i;             *
				*	}                       *
                *****************************
				

2. While-loop

 - 형식 : while(조건문)

 - 사용 예 : 1부터 10까지의 합을 더한다.

				*****************************
				*	i=1   					*
				*	while(i<11){            *
				*		sum+=i;             *
				*		i++	                *
				*	}                       *
                *****************************

3. do-While

 - 형식 : do{}while{조건문}; !!세미콜론 꼭

 - 사용 예 : 1부터 10까지의 합을 더한다.

				*****************************
				*	i=1   					*
				*	do{       	            *
				*		sum+=i;             *
				*		i++	                *
				*	}while(i<11);           *
                *****************************


#Java의 조건문
1. if문

 - 특정 범위의 조건을 명시할 때 사용한다.
 - 사용 예 : 짝수와 홀수 비교
 	
				*****************************
				*	if((num%2)==0)          *
				*		return 0; // 짝수   *			
				*	else				    *
				*		return 1; // 홀수   *
                *****************************


2. switch문
 - if와는 달리 특정값으로 비교한다.
 - 사용 예 : 값 비교
 
				*****************************
				*	switch(num){			*
				*		case 1 : 실행문;	  *
				*				 break;     *
				*		case 2 : 실행문;	  *
				*				 break;		*
				*		case 3 : 실행문;	  *
				*				 break;	    *
				*		default: 실행문;	  *
				*				 break;		*
				*  }						*
				*****************************
